## Rich Music Bot
A music bot with custom playlist support, music downloader, music filters

## Self Hosting
> Configuration
- fill `config.ts` file *some config require `.env`

> Building The Code
- npm install / yarn install
- npm run build / yarn build

> Running The Code
- npm start / yarn start

## Requirements
1. A brain (ofc)
2. Bot Token 
4. Node.js v16.x
